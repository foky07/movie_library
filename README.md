## Использование приложения

1.  **Добавление фильма:**
    Заполните поля «Название», «Жанр», «Год выпуска» и «Рейтинг».
    Нажмите кнопку «Добавить фильм». Фильм появится в таблице.
2.  **Сохранение и загрузка:**
    *   При выходе из программы появится запрос на сохранение.
    *   Данные сохраняются в файл `movies.json` в той же папке.
    *   При следующем запуске программа автоматически загрузит данные из этого файла.
3.  **Проверка ввода:**
    *   Поле «Год» должно содержать число от 1800 до 2100.
    *   Поле «Рейтинг» должно быть числом от 0 до 10.
4.  **Фильтрация:**
    *(Примечание: В текущей версии фильтрация по жанру и году реализована через прямой ввод в поля и кнопку "Добавить")*
